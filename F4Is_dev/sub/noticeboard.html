F4Is 사용방법

1. 페이지 왼쪽 상단의 메뉴바를 누른 후 'F4Is'으로 이동해주세요.
2. "이미지 업로드하기" 버튼을 눌러 이미지를 업로드 해주세요.
3. File Name 란에 자신이 업로드한 이미지가 맞는지 확인 후 "Submit" 버튼을 눌러주세요.
4. AI 모델이 이미지의 음식을 인식하여 해당 음식의 칼로리 정보를 제공합니다.
5. 페이지 왼쪽 상단의 메뉴바를 누른 후 'Notice'으로 이동해주세요.
6. '나이대 별 일일섭취 권장 칼로리량' 제목의 게시물을 눌러 정보를 확인하세요.
7. 일일섭취 권장 칼로리량을 통해 자신에게 맞게 칼로리를 조절하여 건강을 유지합니다.
