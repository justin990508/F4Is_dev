과제를 마치며,

홈페이지개발을 위해 기존에 배웠던 CSS, JAVA, HTML등을 복습하고
모르는 부분을 인터넷에 검색해 홈페이지의 기본틀을 잡았으며, JAVA Script를 활용하여 모션을 넣었습니다.
홈페이지 처럼 첫화면에는 요약정보들을 담았으며, 클릭시 화면을 넘기도록 유도하였습니다.
페이지 구조는 웹진을 통해 구조와 디자인을 분석했습니다. 학기초부터 바로바로 수업을 정리하고 자료를 찾았으며,
홈페이지를 만들기 위해서는 공지가 올라온 후 부터 학습하여 제작하였습니다.

이 과목을 수강하면서 네트워크에 대한 것뿐만 아니라 운영체제, 보안 등의 내용도 모두 이어지는 것을 느끼게 되었습니다.
온라인 수강생인데도 불구하고 교수님의 수업에 대한 열정을 느낄 수 있었고 덕분에 많이 배워가는 것 같습니다!
앞으로도 건강하시고 좋은 교수님의 수업 많이 해주시면 감사하겠습니다.
